# Changelog

## v2.0.0 (2026-02-09)
### Added
- AES-256-GCM encryption with authenticated data
- Argon2id key derivation (replaces PBKDF2)
- Vault locking with auto-lock timer
- Entry categories and tags
- Password generator (passphrase + random)
- Clipboard auto-clear (30s default)
- Audit trail with tamper detection
- Import/Export (JSON, CSV, KeePass XML)
- Search and filter across entries
- REST API for programmatic access
- Comprehensive test suite

## v1.0.0 (2025-12-15)
- Initial release with basic PBKDF2 vault
